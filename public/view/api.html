API

Method : Post

Host : http://g2demo.nxg.me:8080


/morningbird

Request :

(1) auth : {
      service: 'Gmail',
      auth: {
         user: 'gmail.user@gmail.com',
         pass: 'userpass'
      }
}
(2) setting : {
      to: 'mail address 1', 'mail address 2' , ...
      subject: 'mail subject',
      html: 'html code'
      generateTextFromHTML: true || false
}
(3) keys : {
      public: 'your public key',
      private: 'your private key'
}

Response :

- info :

info.messageId :

      most transports should return the final Message-Id value used with this property

info.envelope :

      includes the envelope object for the message

info.accepted :

      is an array returned by SMTP transports (includes recipient addresses that were accepted by the server)

info.rejected :

      is an array returned by SMTP transports (includes recipient addresses that were rejected by the server)

info.pending :

      is an array returned by Direct SMTP transport. Includes recipient addresses that were temporarily rejected together with the server response

response :

      is a string returned by SMTP transports and includes the last SMTP response from the server

- err :

this err is the error object if message failed



Example (by nodejs)

var url : 'http://g2demo.nxg.me/morningbird'
var parms = {
       auth : {
             service : 'Gmail',
             auth : {
                   user : 'gmail.user@gmail.com',
                   pass : 'userpass'
             }
       },
       setting : {
             to: 'mail address 1', 'mail address 2' ,
             subject: 'mail subject',
             html: 'html code',
             generateTextFromHTML: true || false
       },
       keys : {
             public: 'your public key',
             private: 'your private key'
       }
}

$http( { method:'POST' , url:url , data:parms } )
